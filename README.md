# Dotfiles

Instal:
```bash
  # mac os homebrew packages
  - brew install nvim
  - brew install tmux
  - brew install nnn 
  - brew install kitty 

  # tmux plugin manager
  - $ git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Copy:
```
  cp -r ./config ~/.config
```


### Commands:

tmux:

```
  ctrl+a - leader keys

  <leader> c    - create new tab
  <leader> 0-9  - switch to tab
  <leader> |    - split vertical
  <leader> "    - split horizontal
  
  # switch to windows
  <leader> Up
  <leader> Down
  <leader> Right
  <leader> Left
```
